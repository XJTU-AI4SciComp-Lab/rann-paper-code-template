# Paper Code Template

This repository provides a standard template for paper-related code developed by the XJTU AI for Scientific Computing Lab.

It is intended for randomized neural networks, local randomized neural networks, adaptive/growing randomized neural networks, operator learning models, and structure-preserving AI methods for scientific computing.
